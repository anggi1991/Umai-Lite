## ğŸ’» Tech Stack & Features

### **Tech Stack**
- **Frontend:** React Native (Expo 51) + TypeScript
- **UI Framework:** React Native Paper
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **AI:** Azure OpenAI (GPT-4 for chat & tips)
- **Monetization:** RevenueCat (subscriptions) + AdMob (ads)
- **Notifications:** Expo Push Notifications
- **i18n:** 4 languages (Indonesian, English, Japanese, Chinese Simplified)

### **Key Features**
- ğŸ‘¶ **Multi-child support** - Track up to 3 children (Family tier: 5)
- ğŸ“Š **Activity tracking** - Feeding, sleep, diaper, mood, growth
- ğŸ’¬ **AI chat assistant** - Baby Buddy with customizable personality
- ğŸ’¡ **Daily tips** - Personalized parenting advice
- ğŸ”” **Smart reminders** - Recurrence patterns & snooze
- ğŸ“¸ **Photo gallery** - Milestone photos with captions
- ğŸ“ˆ **Growth charts** - Weight, height, head circumference
- ğŸ’ **Freemium model** - Free tier with usage limits
- ğŸ **Referral program** - Earn free months
- ğŸ† **Gamification** - Achievement badges

**Core Tables (11):**
`profiles`, `children`, `activities`, `reminders`, `daily_tips`, `chat_sessions`, `messages`, `media`, `subscriptions`, `notification_logs`, `audit_logs`
