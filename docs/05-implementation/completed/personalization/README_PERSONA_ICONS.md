# AI Persona Mascot Icons

Folder ini berisi 4 gambar maskot untuk AI Personality:

1. **buddy-professional.png** - Maskot dengan ikon chat (untuk persona Professional)
2. **buddy-friendly.png** - Maskot dengan thumbs up (untuk persona Friendly)  
3. **buddy-concise.png** - Maskot dengan ekspresi serius/fokus (untuk persona Concise)
4. **buddy-encouraging.png** - Maskot dengan senyum lebar (untuk persona Encouraging)

## Cara Menambahkan Gambar:

Simpan 4 gambar dari attachment ke folder ini dengan nama file yang sesuai di atas.

## Ukuran yang Disarankan:
- Resolusi: 512x512px atau 1024x1024px
- Format: PNG dengan background transparan
- Aspect ratio: 1:1 (square/circular design)
